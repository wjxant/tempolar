Desarrollo de Aplicaciones Web (DAW)

¿Qué es DAW?

El ciclo formativo de Desarrollo de Aplicaciones Web (DAW) es una titulación de Grado Superior que capacita a los estudiantes para desarrollar, implementar y mantener aplicaciones web. Durante el curso, se adquieren conocimientos técnicos necesarios para el desarrollo tanto en el lado del cliente como del servidor.

¿Qué aprenderás en DAW?

  • Desarrollo en entorno cliente: Programación con lenguajes como HTML, CSS, JavaScript y frameworks modernos como React o Angular.
  • Desarrollo en entorno servidor: Creación de aplicaciones backend utilizando lenguajes y tecnologías como PHP, Java, Python o Node.js.
  • Diseño de interfaces web: Creación de sitios web dinámicos, interactivos y responsivos que se adapten a dispositivos móviles.
  • Bases de datos: Gestión y diseño de bases de datos, así como su conexión con aplicaciones web.
  • Seguridad: Implementación de medidas de seguridad en aplicaciones web para prevenir vulnerabilidades.
  • Despliegue de aplicaciones: Técnicas para la publicación y mantenimiento de aplicaciones web en entornos de producción.

Salidas profesionales

Los titulados en DAW tienen múltiples salidas profesionales en el campo del desarrollo web y tecnológico. Entre las principales oportunidades están:

  • Desarrollador de aplicaciones web.
  • Desarrollador frontend o backend.
  • Administrador de sitios web.
  • Consultor de soluciones web.
  • Especialista en UX/UI.

Requisitos de acceso

Para acceder al ciclo de Grado Superior en DAW, se necesita una de las siguientes titulaciones:

  • Título de Bachillerato.
  • Título de Técnico de Grado Medio.
  • Haber superado la prueba de acceso a ciclos formativos de grado superior.
  • Haber superado la prueba de acceso a la universidad para mayores de 25 años.
