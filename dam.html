Desarrollo de Aplicaciones Multiplataforma (DAM)

¿Qué es DAM?

El ciclo formativo de Desarrollo de Aplicaciones Multiplataforma (DAM) es un título de Grado Superior que prepara a los estudiantes para desarrollar aplicaciones en diferentes plataformas: móviles, de escritorio y otros dispositivos. Se centra en la creación de software que pueda funcionar en diferentes sistemas operativos y entornos.

¿Qué aprenderás en DAM?

  • Programación en múltiples lenguajes: Conocimiento de lenguajes como Java, Kotlin, Swift, C#, y otros que permiten desarrollar aplicaciones multiplataforma.
  • Desarrollo de aplicaciones móviles: Creación de aplicaciones para sistemas operativos móviles como Android e iOS.
  • Desarrollo de aplicaciones de escritorio: Implementación de software para sistemas operativos de escritorio como Windows, macOS y Linux.
  • Uso de bases de datos: Diseño, gestión y conexión de bases de datos con aplicaciones, tanto locales como en la nube.
  • Interfaces de usuario: Diseño y creación de interfaces gráficas para aplicaciones multiplataforma.
  • Seguridad en el desarrollo de aplicaciones: Aplicación de medidas de seguridad en aplicaciones, asegurando la protección de datos y la integridad del software.

Salidas profesionales

Los titulados en DAM pueden acceder a múltiples puestos relacionados con el desarrollo de software y aplicaciones para diversos dispositivos. Algunas de las salidas profesionales incluyen:

  • Desarrollador de aplicaciones multiplataforma.
  • Desarrollador de aplicaciones móviles (Android, iOS).
  • Desarrollador de software de escritorio.
  • Administrador de bases de datos.
  • Especialista en interfaces de usuario (UI/UX).
  • Consultor de desarrollo de software.

Requisitos de acceso

Para acceder al ciclo de Grado Superior en DAM, es necesario cumplir con alguno de los siguientes requisitos:

  • Título de Bachillerato.
  • Título de Técnico de Grado Medio.
  • Haber superado la prueba de acceso a ciclos formativos de grado superior.
  • Haber superado la prueba de acceso a la universidad para mayores de 25 años.
