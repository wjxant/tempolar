¿Qué es la asignatura de Despliegue de Aplicaciones Web?

La asignatura de Despliegue de Aplicaciones Web se centra en los procesos y técnicas necesarios para poner en funcionamiento una aplicación web en un entorno real, disponible para los usuarios finales. A continuación se describen algunos de los temas clave:

  • Configuración de servidores: Aprende a configurar servidores web como Apache o Nginx, y servidores de aplicaciones como Tomcat o Node.js.
  • Virtualización y contenedores: Uso de tecnologías como Docker o Kubernetes para empaquetar y desplegar aplicaciones de manera más eficiente.
  • Gestión de entornos: Creación de entornos de desarrollo, pruebas y producción, y su correcta configuración.
  • CI/CD (Integración y Despliegue Continuo): Automatización del proceso de construcción, pruebas y despliegue de aplicaciones utilizando herramientas como Jenkins, GitLab CI, o GitHub Actions.
  • Gestión de bases de datos: Despliegue de bases de datos en servidores y configuración para garantizar la disponibilidad y rendimiento.
  • Escalabilidad y balanceo de carga: Técnicas para asegurar que la aplicación web pueda manejar una gran cantidad de usuarios simultáneamente.
  • Seguridad: Implementación de certificados SSL/TLS, configuraciones de seguridad para servidores y prácticas de seguridad web.
